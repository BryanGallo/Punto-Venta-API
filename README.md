## BagoShop

1. Clonar Proyecto
2. ``npm install``

3. Instalar modulos
```bash
$ npm install
```

4. Arrancar el proyecto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
