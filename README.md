## BagoShop

1. Clonar Proyecto
2. ``npm install``
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno

5. Instalar modulos
```bash
$ npm install
```

6. Arrancar el proyecto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
